Q: What does DBMS stand for?
A: Data Binary Management System
B: Database Management System
C: Data Backup Management System
D: Data Basic Management System
ANSWER: B

Q: Which of the following is a type of DBMS?
A: Hierarchical
B: Network
C: Relational
D: All of the above
ANSWER: D

Q: Which SQL statement is used to retrieve data from a database?
A: SELECT
B: GET
C: RETRIEVE
D: FETCH
ANSWER: A

Q: Which key uniquely identifies a record in a table?
A: Foreign Key
B: Primary Key
C: Candidate Key
D: Composite Key
ANSWER: B

Q: Which SQL statement is used to remove all records from a table without deleting the table itself?
A: DELETE
B: DROP
C: TRUNCATE
D: REMOVE
ANSWER: C

Q: In a relational database, a relation is also called a:
A: Field
B: Tuple
C: Table
D: Attribute
ANSWER: C

Q: Which constraint ensures that a column cannot have NULL values?
A: UNIQUE
B: NOT NULL
C: PRIMARY KEY
D: FOREIGN KEY
ANSWER: B

Q: What is the purpose of a foreign key in a table?
A: To uniquely identify each record in the table
B: To create a link between two tables
C: To prevent NULL values
D: To sort data in ascending order
ANSWER: B
